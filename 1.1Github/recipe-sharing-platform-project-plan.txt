âš™ï¸ GitHub Actions (CI/CD)

Workflow File: .github/workflows/ci.yml

    name: CI Pipeline

    on: [push, pull_request]

    jobs:
      build:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v3
          - name: Validate HTML
            run: |
              npm install -g htmlhint
              htmlhint "**/*.html"
          - name: Validate CSS
            run: |
              npm install -g stylelint
              stylelint "**/*.css"
          - name: Validate JS
            run: |
              npm install -g eslint
              eslint "**/*.js"

Outcome: Every time you push code or create a PR, GitHub automatically
checks your HTML, CSS, and JS for errors.
